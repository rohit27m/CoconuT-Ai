## Using the Platform

### 1. Emotion AI Analysis
- Click **"Initialize Emotion AI"** to start camera
- Position your face in the frame
- Click **"Analyze Emotion"** to detect mood
- Or click **"Skip Analysis"** to continue

### 2. Chat Interface
- Enter your message in the input field
- AI will show thinking animation while processing
- Receive intelligent, mood-aware responses
- View session analytics in the Analytics Dashboard

### 3. Ambient Audio System
- **Ambient Sounds**: Adjusts to your emotional state
- **Music Player**: Play the BIBA track or add more songs
- **Volume Controls**: Independent sliders for ambient and music
- **Now Playing**: See current track information

### 4. Analytics Dashboard
- Click **"Analytics"** button in header
- View total sessions
- See data points learned
- Check your user profile

## Troubleshooting

### If the page doesn't load:
1. Ensure Flask is running (check terminal)
2. Verify URL: http://127.0.0.1:5000
3. Check firewall settings
4. Try http://localhost:5000

### If camera doesn't work:
1. Grant browser permissions
2. Check if another app is using camera
3. Use "Skip Analysis" to continue

### If audio doesn't play:
1. Click toggle buttons to enable
2. Check browser autoplay settings
3. Ensure volume isn't muted
